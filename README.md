# Naturalization Civics Study

A 100% client-side React app for studying the 2025 USCIS naturalization civics test.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The app uses `data/civics-questions-2025.json` as its local data source and deploys to GitHub Pages from `main`.
